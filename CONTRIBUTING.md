# Contributing to Create: Electro Energetics

Everyone is welcome to contribute - whether to add a feature, fix something small, or create a translation.

If you want to contribute, feel free to open a PR or send me a git patch directly.

### Creating translations
1. Take the English localization from [`src/generated/resources/assets/electroenergetics/lang/en_us`](https://github.com/george8188625/Create-Electro-Energetics/blob/1.21.1/src/generated/resources/assets/electroenergetics/lang/en_us.json)
2. Place it in [`src/main/resources/assets/electroenergetics/lang/`](https://github.com/george8188625/Create-Electro-Energetics/tree/1.21.1/src/main/resources/assets/electroenergetics/lang) with the correct name ( e.g. `ru_ru` for russian )
3. Translate the entries
4. Create a pull-request with the translation file

If you want to update an existing translation, do the same, but instead of copying the file, add missing entries to the already existing file.

### What to do

If you want to contribute, but not sure what, then check out [`checklist.md`](https://github.com/george8188625/Create-Electro-Energetics/blob/1.21.1/checklist.md)
